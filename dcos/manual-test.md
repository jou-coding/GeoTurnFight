# 手動テスト・動作確認仕様書

本ドキュメントは、本アプリを手動で動作確認するためのチェック項目です。
自動テストコードは作成していませんが、開発者自身による確認を実施しています。

## 1. 基本動作確認

- [ ] トップページが表示される
- [ ] ユーザー名を入力して入室できる
- [ ] 2人が同じ部屋に参加できる
- [ ] ゲーム画面に正しく遷移する


## 2. バリデーションチェック

- [ ] ユーザー名が空の場合、エラーになる
- [ ] 国名が空文字の場合、送信できない
- [ ] すでに使用された国名は入力できない
- [ ] 想定外の文字列でもアプリが落ちない

## 3. 状態遷移の確認

- [ ] プレイヤー1 → プレイヤー2へターンが切り替わる
- [ ] 自分のターン以外では入力できない
- [ ] 勝敗が決定したら入力が無効になる
- [ ] 勝敗後に結果表示が行われる

## 4. UI・表示確認

- [ ] スマートフォン幅（375px）でもレイアウトが崩れない
- [ ] ボタンや入力欄が画面外にはみ出ない
- [ ] エラーメッセージが視認できる
- [ ] ターンが視覚的に分かる

## 5. デプロイ後の確認

- [ ] 本番URLにアクセスできる
- [ ] 初回アクセス時にエラーが出ない
- [ ] ローカル環境と同様に動作する
